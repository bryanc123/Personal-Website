About Me

Hello, my name is Bryan Cayanan.

I graduated from the Angeles University Foundation in the Philippines back in 2015 and received a bachelor's degree in information technology, but I was not really interested in web development until a couple of years later.

At my university, IT students studied only the basics of web development. We learned HTML, CSS, JavaScript, and some PHP, but the bulk of our programming studies centered around writing desktop applications using Java or C#.

Since web development was not the main focus in my college, I am mostly self-taught as far as web development goes. Articles, tutorials, books, documentation, and other people's code served as learning references.

Immediately after college, I did not have any luck landing jobs as a programmer, and due to personal circumstances, I had to step away from programming for a while. Eventually, I started learning programming again and turned my attention to web development because of the remote work opportunities that were available.

It is my goal to become a full-stack developer, but I am primarily interested in frontend development. However, if I find employment as a web developer, then I am willing to learn any kind of technology, frontend or backend. Web technologies are always evolving, so I am always interested in learning more about them while maintaining my current skills.

Thank you for reading.
